# Input

Inputs allow users to enter and edit text. They include support for labels and error states.

## Usage

<DemoBlock>
  <div className="w-full max-w-sm space-y-4">
    <Input label="Email" placeholder="Enter your email" type="email" />
    <Input label="Password" defaultValue="secret" type="password" error="Invalid password provided" />
  </div>
</DemoBlock>

<CodeBlock 
  language="tsx" 
  code={`import { Input } from '@/components/ui/input';

export default function Example() {
  return (
    <div className="space-y-4">
      <Input label="Email" placeholder="Enter your email" type="email" />
      <Input label="Password" type="password" error="Invalid password" />
    </div>
  );
}`} 
/>
